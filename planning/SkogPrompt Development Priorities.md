# SkogPrompt Development Priorities

## Core Goals [x]

- **Primary Purpose**: Create prompts dynamically and on demand
- **Short-term Goal**: Create necessary context needed for AI agent chats (like we have currently)
- **Long-term Goal**: Advanced functionality to be determined

## Immediate Priorities [x]

1. **Integration with SkogAI Tools** - HIGHEST PRIORITY
   - Focus on connecting with the broader SkogAI ecosystem
   - Enable seamless workflows between tools

2. **Template System Implementation**
   - Approach depends on quality and speed of implementation
   - Can build on skogmcp components or create new approaches as appropriate
   - No predetermined path; focus on what works best

3. **Existing Functionality Enhancement**
   - Improve current prompt execution via gptme
   - Strengthen chat message conversion capabilities
   - Enhance context generation

## Development Approach [x]

- No specific AI platforms as initial focus for format conversion
- Success metrics based on delivering agreed functionality for each release
- Adaptive development based on real-world usage and needs
- Exploratory rather than fixed roadmap

## Current Status [x]

- Basic prompt execution working
- Output saving to files implemented
- Simple context generation functional
- Chat message conversion to JSON available
- Architectural foundations under development

## Next Steps [/]

1. Further define integration points with other SkogAI tools
2. Develop initial template system implementation
3. Document integration architecture
4. Create minimal viable functionality for template-based prompt generation

## Documentation Status [x]

1. README updated with current functionality and direction
2. Architectural documentation created:
   - SkogPrompt Integration with Template-Based Prompt Generation
   - SkogPrompt Templating System Design
